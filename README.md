## 🚀 Deployment Guide

### Step 1 — Create a GitHub Repository

1. Go to [github.com/new](https://github.com/new)
2. Name it `mrcp-guide-2026` (or any name you prefer)
3. Set it to **Public** (required for free Vercel deployment)
4. Click **Create repository**

---

### Step 2 — Upload the Files

#### Option A: GitHub Web Interface (easiest)
1. In your new repository, click **Add file → Upload files**
2. Drag and drop these files:
   - `index.html`
   - `vercel.json`
   - `README.md`
3. Click **Commit changes**

#### Option B: Git Command Line
```bash
git clone https://github.com/YOUR_USERNAME/mrcp-guide-2026.git
cd mrcp-guide-2026

# Copy your downloaded files into this folder, then:
git add .
git commit -m "Initial commit: MRCP guide 2026"
git push origin main
```

---

### Step 3 — Deploy to Vercel

1. Go to [vercel.com](https://vercel.com) and sign up / log in with your GitHub account
2. Click **New Project**
3. Find `mrcp-guide-2026` in your repository list → click **Import**
4. Leave all settings as default — Vercel auto-detects static HTML
5. Click **Deploy**
6. ✅ Your site is live in ~30 seconds!

---

### Step 4 — (Optional) Custom Domain

1. In your Vercel project → **Settings → Domains**
2. Add a custom domain like `mrcp2026.yourdomain.com`
3. Follow Vercel's DNS instructions for your domain registrar

## 🧠 Study Plan Summary

| Phase | Period | Target Exam | Focus |
|---|---|---|---|
| Phase 1 | Feb → May 2026 | May/June diet | MRCP Part 1 |
| Phase 2 | Jun → Aug 2026 | Sept/Oct diet | MRCP Part 2 Written |
| Phase 3 | Oct → Dec 2026 | Dec diet | PACES |

**Weekly rhythm:** 5 days question practice + 1 day weak topic revision + 1 day rest  
**Daily target (Phase 1 & 2):** 100–120 questions with explanation review  
**Monthly hours:** ~60–70 hrs → ~650 total over 10 months
